METHODS：公共方法

组件的公共方法

给 Keystone 增加 show、hide 方法：


    Keystone.METHODS = {
        show:function () {
            this.get('el').fadeIn();
        },
        hide:function () {
            this.get('el').fadeOut();
        }
    };
当前上下文（this）是当前组件的实例对象

亲，还有一件事情很重要，将 Keystone.METHODS 成员复制到 Keystone.prototype 上。


    S.augment(Keystone, Keystone.METHODS);

上一步很重要，目的就是把这些组件方法开放出去供组件实例使用。至于具体组件内部需要提供什么方法，就要视组件不同而不同了。
